# Project 1 - MyStory

Submitted by: Sabrina Zheng

MyStory is an app that introduces a simplified profile page about Spiderman The. initial screen of this app has a few high-level context items about the person/thing. Users are able to tap into those items to reveal more information about them.

Time spent: 5 hours spent in total
